About the Courses

1. Web Development

This course introduces students to the fundamentals of web design and development. You will learn HTML, CSS, and JavaScript, along with practical projects to build real websites. By the end of this course, students will be able to create and host their own responsive websites.

2. Graphic Design

This program covers the basics of visual communication using popular design software. Students will learn color theory, typography, and layout techniques used in digital and print design. The course also includes portfolio creation to help students showcase their work professionally.

3. Database Management

The Database Management course provides an introduction to relational database systems, SQL commands, and data modeling. Students will learn how to design, build, and manage databases using MySQL and PostgreSQL.

Learning Outcomes

  • Understand theoretical and practical aspects of modern IT skills.
  • Develop projects that demonstrate creativity and technical proficiency.
  • Work independently and collaboratively in digital environments.
  • Build a solid foundation for further studies or employment.
